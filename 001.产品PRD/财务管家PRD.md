# 财务管家 V1.0.1 - 产品需求文档（PRD）

---

## 1. 产品概述

**产品名称**：财务管家
**产品类型**：个人财务记录与管理工具（Web端）
**一句话描述**：帮助用户通过手机浏览器轻松记录每日消费、分类统计支出、清晰了解财务状况。

**核心价值**：
- 随时随地快速记账
- 了解钱花在哪里
- 培养良好消费习惯

---

## 2. 目标用户

**主要用户群体**：
- 个人用户：18岁以上，有日常消费记录需求的人群
- 家庭用户：家庭成员共同记录生活开支
- 情侣用户：情侣双方共同记录共享消费

**扩展用户群体**：
- 小微企业：小型创业团队记录运营成本
- 个体户：记录店铺日常收支流水

## 3. 用户故事

| 角色 | 场景 | 需求 |
|------|------|------|
| 个人用户-小明 | 每天下班买菜 | 2分钟内记录今日支出 |
| 个人用户-小红 | 月底查看报表 | 清楚知道本月花销分布 |
| 上班族 | 管理日常开支 | 设置预算，控制不超支 |
| 家庭用户-夫妻 | 共同记录家庭开支 | 了解家庭整体收支情况 |
| 情侣用户 | 记录约会开销 | 分摊共同消费 |
| 个体户-淘宝店主 | 记录店铺每日收入支出 | 了解经营状况 |
| 小微企业-创业团队 | 记录团队运营成本 | 控制开支 |

---

## 3. 功能需求

### 3.1 核心功能（必做）

#### 3.1.1 用户认证
- **注册功能**：用户名 + 密码注册
- **登录功能**：用户名 + 密码登录
- **登录状态**：登录后自动保持会话，刷新页面不丢失登录状态
- **登出功能**：用户可主动退出登录
- **数据隔离**：用户只能查看自己的数据，不显示其他用户数据

#### 3.1.2 记账功能
- **快速记账**：选择类型（支出/收入）→ 选择分类 → 输入金额 → 保存（4步完成）
- **支持收支**：支持记录支出和收入两种类型
- **支持平台**：Web端（手机浏览器优先支持），适配移动端浏览器
- **记账字段**：
  - 类型（必填：支出/收入）
  - 金额（必填）
  - 分类（必填）
  - 日期（默认当天，可修改）
  - 备注（选填）
  - 支付方式（选填：现金/支付宝/微信/银行卡/对公账户）

#### 3.1.3 支出分类
预设常用分类（可根据需要自定义）：

| 一级分类 | 二级分类示例 |
|----------|--------------|
| 餐饮 | 早餐、午餐、晚餐、零食、奶茶 |
| 交通 | 公交、地铁、打车、停车、油费 |
| 购物 | 衣服、数码、日用品、化妆品 |
| 居住 | 房租、水电费、物业费、维修 |
| 娱乐 | 电影、游戏、旅游、演出 |
| 医疗 | 药品、医院、检查 |
| 教育 | 课程、书籍、培训 |
| 通讯 | 手机费、网络费 |
| 人情 | 礼物、红包、人际交往 |
| 其他 | 捐赠、宠物等 |
| 营业收入 | 产品销售、服务收入（个体户/小微） |
| 进货成本 | 原材料采购、商品进货（个体户/小微） |
| 运营支出 | 租金、工资、水电费（个体户/小微） |

#### 3.1.4 流水记录
- 按日/周/月展示消费记录
- 支持搜索特定日期或备注关键词
- 可对单条记录进行编辑/删除

#### 3.1.5 统计报表
- **月度统计**：本月支出总额、各分类金额及占比
- **趋势图表**：近3个月/6个月的支出趋势折线图
- **分类饼图**：直观展示各分类支出占比

#### 3.1.6 预算管理
- 设置月度总预算
- 显示剩余可用预算
- 预算超支提醒（红色警示）

## 4. 非功能需求

### 4.1 性能要求
- 记账操作响应时间 < 1秒
- 启动时间 < 3秒
- 支持离线记账（网络恢复后自动同步）

### 4.2 数据安全
- 本地数据加密存储
- 支持数据备份（云端/本地）

### 4.3 兼容性
- 移动端浏览器：Safari（iOS）、Chrome（Android）、Edge
- 桌面浏览器仅作预览参考（主要面向手机浏览器使用）
- 响应式布局，适配主流手机屏幕尺寸（320px - 428px宽度）

## 6. UI设计规范

### 6.1 视觉风格
- **风格定位**：可爱、温暖、轻松
- **设计关键词**：亲和力、简洁、温暖

### 6.2 色彩规范
| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | #FF9EAA | 粉嫩粉色，用于主按钮、强调元素 |
| 辅助色 | #FFB6C1 | 浅粉色，用于卡片背景、hover状态 |
| 强调色 | #FF6B8A | 玫瑰红，用于重要提示、支出警示 |
| 收入色 | #7DD87D | 清新绿色，用于收入相关展示 |
| 背景色 | #FFF5F7 | 极浅粉色，页面主背景 |
| 卡片背景 | #FFFFFF | 白色 |
| 文字主色 | #4A4A4A | 深灰色 |
| 文字辅色 | #999999 | 浅灰色 |

### 6.3 圆角规范
- 按钮圆角：20px（大圆角）
- 卡片圆角：16px
- 输入框圆角：12px

### 6.4 字体规范
- 主字体：系统默认字体，清晰易读
- 数字字体：等宽字体展示金额
