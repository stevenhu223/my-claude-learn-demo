# CLAUDE.md

本文件为 Claude Code (claude.ai/code) 在本仓库工作时提供指导。

## 项目概述

这是一个名为"财务管家"的个人财务管理Web应用。项目目前处于早期开发阶段。

**主要组件**：
- [财务管家PRD.md](001.产品PRD/财务管家PRD.md) - 产品需求文档
- [财务管家/index.html](财务管家/index.html) - 完整的Web应用（单文件，移动端优先）

## 架构

应用为单页HTML架构：
- 纯 HTML/CSS/JS（无框架）
- LocalStorage 持久化数据
- 移动端优先的响应式设计
- 粉色可爱风格主题

## 开发

无需构建流程。直接在浏览器中打开 `财务管家/index.html` 即可运行。

## 未来开发计划

当前为 Web 端 V1.0 版本。后续迭代计划：
- V2.0：多设备同步、共享账本
- V3.0：智能分析、消费建议
- V4.0：打通支付平台、自动记账
